# AI测试平台 - 更新日志

## [2024-03-09] 初始版本

### 新增功能

#### 1. 项目管理
- 创建/查看/删除项目
- 项目分组管理测试用例

#### 2. 需求文档管理
- 支持 PDF 文档上传和解析
- 支持 Word (.docx) 文档上传和解析
- 自动提取文档文本内容

#### 3. AI 测试用例生成
- 基于需求文档自动生成测试用例
- 支持人工审核和编辑
- 批量保存测试用例

#### 4. 测试用例管理
- 创建/查看/编辑/删除测试用例
- 按优先级筛选
- 关键词搜索
- 批量操作（执行、删除）

#### 5. 测试执行
- 单个/批量执行测试用例
- 基于 Agent 的自动化测试
- 实时进度推送 (WebSocket)
- 自动截图和 GIF 动画生成

#### 6. 数据库
- SQLite 数据库存储
- 完整的数据模型设计
- 数据持久化

### 技术栈

#### 后端
- FastAPI - Web 框架
- Pydantic - 数据验证
- WebSocket - 实时通信
- PyPDF2 - PDF 解析
- python-docx - Word 解析

#### 前端
- Vue 2.7 - 前端框架
- Element UI - UI 组件库
- Axios - HTTP 请求

### API 接口

| 模块 | 端点数量 | 说明 |
|------|----------|------|
| 项目管理 | 5 | CRUD 操作 |
| 文档管理 | 4 | 上传、解析、查询 |
| 测试用例 | 9 | 完整的用例管理 |
| 测试执行 | 6 | 执行和监控 |
| 测试报告 | 3 | 报告查询 |

### 数据库表

| 表名 | 说明 |
|------|------|
| projects | 项目表 |
| documents | 需求文档表 |
| test_cases | 测试用例表 |
| test_executions | 测试执行记录表 |
| test_reports | 测试报告表 |
| system_configs | 系统配置表 |
